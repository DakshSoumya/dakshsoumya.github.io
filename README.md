# JEE Tracker App

A comprehensive productivity and analytics tool designed for students preparing for the Joint Entrance Examination (JEE). This application enables users to systematically track syllabus completion, analyze performance, and optimize revision strategies through data-driven insights.


TO USE : https://dakshsoumya.github.io
---

## 🚀 Features

### 📊 Syllabus Tracking
- Structured tracking of subjects, chapters, and subtopics  
- Progress visualization using percentage completion and heatmaps  
- Categorization of topics based on proficiency (Strong / Weak / Needs Revision)

---

### 📘 Previous Year Questions (PYQ) Integration
- Topic-wise mapping of JEE Main and JEE Advanced questions  
- Tracking of solved questions and accuracy  
- Performance insights for each topic

---

### 🧠 Mistake Log System
- Record and classify mistakes:
  - Conceptual Errors  
  - Calculation Errors  
  - Careless Mistakes  
- Weekly performance summaries to identify recurring weaknesses  

---

### 🔁 Intelligent Revision Scheduler
- Automated revision planning based on spaced repetition  
- Custom revision intervals (1 day, 3 days, 7 days, etc.)  
- Prioritization of weak topics  

---

### ⏱️ Study Analytics Dashboard
- Daily study time tracking  
- Efficiency analysis  
- Visual representation of study consistency and trends  

---

### 📝 Mock Test Analysis
- Input and evaluate mock test results  
- Subject-wise and topic-wise performance breakdown  
- Accuracy and time management insights  

---

### 📈 Rank Predictor
- Estimate JEE Main percentile and rank based on mock scores  
- Performance trend tracking over time  

---

### 🎯 Focus Mode
- Distraction-free study timer  
- Optional ambient sounds (e.g., white noise, rain)  
- Session tracking for productivity analysis  

---

### ⚠️ Reality Check System
- Provides critical feedback on study pace and consistency  
- Highlights neglected subjects and incomplete syllabus areas  

---

### 🔗 Peer Comparison (Optional)
- Compare study metrics with peers  
- Analyze relative performance and consistency  

---

### 🔥 Streak Tracking
- Monitor daily study consistency  
- Encourage habit formation through streak-based tracking  

---

## 🎨 UI/UX Design

- Minimalist interface with a dark theme  
- Neon green highlights for key elements  
- Graph-focused dashboard for clarity and usability  
- Fully responsive design for desktop and mobile devices
